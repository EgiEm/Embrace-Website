# 💼 Andi's Personal Developer Portfolio

An interactive and fully responsive personal portfolio website showcasing developer achievements, skill sets, background timelines, and completed projects.

## 🚀 Features

- **Modern Grid & Flexbox Layout**: Clean layout architecture with responsive breakpoints.
- **Professional Timeline**: Elegant step-by-step history showing professional progression and training.
- **Interactive Card Elements**: Interactive cards with micro-animations highlighting project scopes.
- **Contact Setup**: Front-facing contact panel and developer details.

## 🛠️ Technologies Used

- **HTML5** - Structuring page elements
- **CSS3** - Style layouts and animations
- **JavaScript** - Interactive scrolling and visual transitions
